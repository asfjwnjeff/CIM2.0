### 2.2 模块一：审批字段配置（新增）

**入口**：侧栏 → 风控审批 → 字段配置（`/approval/fields`）

核心功能：
- 审批结构化字段池，支持 8 种字段类型：数值下拉、百分比、单选、多选、单选+其他、布尔、接口查询、文本输入
- 每个字段可绑定一个或多个服务产品
- 可选值配置支持增删改排序
- 完整 CRUD 操作（前端 + API + 数据库表）
- 13 条初始种子数据，覆盖运输/货代/仓库等场景

### 2.3 模块二：审批流模板（改造）

**入口**：侧栏 → 风控审批 → 流程配置（`/approval/workflows`）

核心功能：
- 统一 6 级审批节点结构：发起人(1) → 部门经理(2) → 职能审批人(3) → 财务审批会签(4) → 总经理(5) → IT运维确认(6)
- 7 个审批流模板（对应 7 个服务产品）
- 职能审批节点支持动态审批人 + 条件分支追加
- 列表页支持搜索、筛选、复制模板
- 编辑页 6 级节点可视化配置

### 2.4 模块三：审批申请页重构

**涉及页面**：新建 `/approvals/new`、编辑 `/[id]/edit`、详情 `/[id]`、列表 `/approvals`

核心升级：
- **双栏布局**：左侧表单（4 组字段分区 + 色标），右侧审批流实时预览
- **动态字段联动**：选服务产品 → 左侧动态字段自动切换 → 右侧审批流模板实时匹配
- **字段分组**：基础信息(蓝) / 业务评估(橙) / 合规审核(绿) / 结算与合同(灰)
- **视觉标识**：动态字段浅蓝底色（`#F0F1FF`），贸易代理=是时浅黄底色（`#FFF9EB`）
- **审批流可视化**：提取 `ApprovalFlowVisual` 组件，支持 preview/progress 双模式

### 2.5 审批辅助报告组件

**组件**：`ApprovalReport.tsx`

- 紧凑表格布局：审批点 / 字段 / 填写值 / 判断条件
