🔄 工作流程

📦 账单拆分流程

1. 获取字段配置

COS调用CIM接口获取客户的账单区分字段配置

2. 用户录入字段值

用户在COS中录入订单及账单区分字段值（Plant、Location、客户部门等）

3. 发送订单到CIM

COS将订单号、客户ID、字段值发送给CIM

4. 规则匹配

CIM按优先级从高到低匹配BillingRule，首次匹配成功即返回

5. 返回账单主体

CIM返回匹配的BillingEntity给COS

6. 记录映射结果

CIM记录OrderMapping，CPQ可查询获取账单主体信息

⚙️ 规则配置流程

1. 选择客户

在账单主体规则管理中选择要配置规则的客户

2. 创建规则

点击"新增规则"，填写规则名称和优先级

3. 配置条件组

添加条件组，支持嵌套AND/OR逻辑，设置字段、操作符、值

4. 选择目标主体

选择规则匹配成功后的目标账单主体

5. 保存并激活

保存规则并设置为激活状态，规则立即生效
