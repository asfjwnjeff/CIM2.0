# 风控审批流程增强 - 实现计划

## 概述

在现有的风控审批系统基础上，增强审批流程展示和规则配置功能，支持是否涉及贸易代理的自动规则处理。

## 技术方案

| 维度 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Next.js 16 + React 19 | 项目已有架构 |
| 状态管理 | React Context + useReducer | 项目已有方案 |
| 样式方案 | Tailwind CSS 4 + 项目现有UI标准 | 统一视觉风格 |
| 类型安全 | TypeScript | 项目已有类型体系 |

## 功能模块

### 1. 风控审批详情页增强 (`/app/approvals/[id]/page.tsx`)

**核心功能**：
- 在页面下方展示审批流程展示区
- 展示当前审批的完整审批节点信息
- 每个节点展示：发起人→部门经理→职能审批人→财务审批人→各中心负责人/总经理→IT运维确认
- 显示节点状态：待审批、已通过、已驳回
- 审批人信息展示

### 2. 风控审批编辑页增强 (`/app/approvals/[id]/edit/page.tsx` 和 `/app/approvals/new/page.tsx`)

**核心功能**：
- 添加"是否涉及贸易代理"选择器
- 选择"是"时，审批流程预览区自动展示包含白沥节点
- 实时预览当前配置的审批流程

### 3. 风控审批列表页增强 (`/app/approvals/page.tsx`)

**核心功能**：
- 列表展示新增列：是否涉及贸易代理（是/否）
- 新增筛选器：是否涉及贸易代理（全部/是/否）
- 卡片视图和表格视图都同步更新

### 4. 审批流程配置页增强 (`/app/approval/workflows/[id]/page.tsx` 和 `/app/approval/workflows/new/page.tsx`)

**核心功能**：
- 新增规则配置模块
- 支持创建/编辑/删除规则
- 规则条件：是否涉及贸易代理 = 是
- 规则动作：自动添加白沥节点到审批流程
- 规则状态：启用/禁用开关
- 规则列表展示：规则名称、条件、动作、状态、操作

**数据结构示例**：
```typescript
{
  id: 'rule-001,
  name: '贸易代理自动加白沥',
  condition: {
    field: 'isTradeAgent',
    operator: 'equals',
    value: true
  },
  action: {
    type: 'add_approver',
    approver: { id: 'user-baili', name: '白沥', role: '职能审批人' }
  },
  enabled: true,
  createdAt: '2026-05-15',
  updatedAt: '2026-05-15'
}
```
